What is CI/CD?

CI/CD is a modern software development practice that automates the process of delivering applications. Continuous Integration (CI) automatically builds and tests code changes whenever developers push to the repository, catching issues early. Continuous Deployment (CD) automatically deploys the approved code to production. This automation accelerates delivery, reduces manual errors, and enables teams to release updates more frequently and reliably.

My Pipeline

When I push code to the main branch on GitHub, GitHub Actions automatically triggers my workflow. The workflow first checks that all required files (index.html, style.css, script.js) exist, then uploads the website artifacts. Finally, it deploys the site to GitHub Pages, making the updates live within minutes without any manual intervention.
